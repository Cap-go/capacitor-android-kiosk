# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-06

### Added
- Initial release of Android Kiosk Mode plugin
- Enter/exit kiosk mode functionality
- Launcher detection and configuration
- Hardware key control (allow/block specific keys)
- Support for Android 6.0 (API 23) through Android 15 (API 35)
- TypeScript definitions with full documentation
- Web platform stub implementation
- iOS stub implementation with Guided Access instructions
- Comprehensive README with usage examples
